# SmartCAPI Backend

SmartCAPI (Smart Computer Assisted Personal Interviewing) is a Progressive Web Application that combines speaker recognition, speech-to-text, and Large Language Model (LLM) to assist in personal interviews.

## Features

- Speaker recognition using Random Forest with 33 MFCC features
- Speech-to-text using Whisper model
- Information extraction using OpenAI API
- Real-time audio processing via WebSocket
- Offline-first architecture with sync capabilities
